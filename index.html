About me

こんにちは！札幌でウェブ制作を学んでいる狩集(かりしゅう)と申します。
自動車産業大国・愛知県生まれ、アメリカのど田舎・ケンタッキー育ちです。

大学卒業後、豊田市で自動車生産設備営業として働いたのち、海外に3年ほど移住し、ツアーガイドやシェフ・ブロッコリー農家を経験しました。 日本に帰国したあと、ずっと興味のあったウェブ開発について勉強をはじめました。 日々勉強の傍、地域の小さなお店やキャンプ場などのウェブサイト制作や、チラシや名刺制作などを行っています。

ウェブ開発を本格的に学び始め、頭の中にあるアイディアがデザイン・コーディングを通してウェブサイトという形になることに感動を覚え、 ウェブ業界に就職してもっと学びたいと感じています！

その他の趣味特技・好きなこと：
・海外旅行
・スノーボード
・翻訳/通訳

HTML

SASSを使用し、CSSでのレスポンシブ対応を考慮したHTMLの構築ができます。使用エディタはVisual Studio Codeです。

CSS

SASSを使用し、レスポンシブ対応したCSSの構築ができます。使用エディタはVisual Studio Codeです。

Photoshop

写真の切り抜きや修正、加工などの作業が可能です。ウェブサイト用の写真だけでなく、名刺やチラシなどの制作の実績もあります。

AdobeXD

ワイヤーフレーム、デザインカンプの作成が可能です。プロトタイプやコンポーネントを使って動きをつけることもできます。

jQuery

スライダー、クリック・スクロール効果、フェードインなどのウェブサイトの動きをつけるためのjQueryコードのカスタマイズが可能です。

Wordpress

無料/有料のテーマをカスタマイズしてウェブサイト構築することが可能です。また、自作テーマでのウェブサイト制作もできます。

Git

自分のソースコードをGitHub上で管理しています。またGithub Pagesを利用してサーバーに自分の作品を掲載しています。

Food Truck ゆらぎ

ニセコのお惣菜フードトラックのウェブサイト

制作時間：55時間
作業環境：WordPress, Visual Studio Code, Photoshop, AdobeXD
使用言語：HTML, CSS, jQuery, PHP
レスポンシブ：対応
サイト言語：日本語・英語

ニセコを中心に北海道各地で出店しているフードトラックのウェブサイトです。
オーナーの希望により白の背景を使いシンプルに、かつjQueryで動きをつけ、モダンなデザインで仕上げました。 地元のお年寄りから若い外国人まで広い層をターゲットに、日本語と英語ページがあります。

はじめにAdobeXDでワイヤーフレームとデザインカンプを制作、Visual Studio Codeでコーディングを行い、 最終的にWordPressの自作テーマとして公開しました。
サーバーはAWSを使用しています。
